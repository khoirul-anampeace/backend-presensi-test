# backend-presensi

Repositori ini berisi backend sederhana untuk aplikasi presensi. Dibangun menggunakan Node.js dan Express, aplikasi ini dapat digunakan untuk mencatat kehadiran pengguna secara digital.

## Fitur Utama

- API untuk manajemen data presensi
- Otentikasi pengguna (jika tersedia)

## Cara Instalasi

1. Clone repositori ini:
   ```
   git clone https://github.com/khoirul-anampeace/backend-presensi-test.git
   ```
2. Masuk ke folder proyek:
   ```
   cd backend-presensi-test
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Cara Menjalankan

Jalankan perintah berikut untuk memulai server:

```
npm start
```

Server akan berjalan di port default (5000 seusai di environment). Ubah konfigurasi sesuai kebutuhan.
